# Thư mục Thesis

Thư mục này chứa các tài liệu liên quan đến luận văn:

## Nội dung
- File Word (.docx) - Báo cáo luận văn
- File PowerPoint (.pptx) - Slide thuyết trình

## Hướng dẫn
1. Đặt file Word báo cáo luận văn vào đây
2. Đặt file PowerPoint slide thuyết trình vào đây
3. Đặt tên file rõ ràng, ví dụ:
   - `BaoCaoLuanVan_LieuKienAn.docx`
   - `SlideThuyetTrinh_LieuKienAn.pptx`
